# 波场 (TRON) 哈希走势实时分析大盘 (路图增强版)

这是一个专为波场区块链设计的专业级哈希数据可视化工具。它通过连接 TronGrid API，实时捕获区块哈希，并自动生成博弈级的可视化路图（大路、珠盘路），帮助用户深度分析哈希末位数字的“单双”与“大小”走势。

## 核心功能

- **全方位可视化路图**：
  - **大路 (Big Road)**：经典趋势追踪，连续结果自动成列，支持自定义行数（3-12行）。
  - **珠盘路 (Bead Road)**：按出块顺序精确填充的矩阵，还原历史全貌，支持独立行数配置。
- **2x2 深度分析面板**：
  - **单双模块**：红蓝经典配色，实时统计单双比例。
  - **大小模块**：橙紫专业配色，实时统计大小分布。
- **高性能实时监听**：
  - **3秒极速同步**：毫秒级监听波场主网出块，实时推送。
  - **内存级缓存**：支持数千个区块的流畅滚动与快速检索。
- **高级采样逻辑**：
  - **自定义间隔**：支持 1, 20, 60, 100 等任意区块采样步长。
  - **偏移对齐**：支持锁定起始区块高度，实现特定规则的对齐分析。
- **交互增强**：
  - **区块跳转**：点击列表任意行，直接跳转至 Tronscan 查看区块详情。
  - **实时搜索**：支持按区块号或 Hash 全文检索，且不干扰走势图的连续性。

## 技术栈

- **React 19**：采用 Concurrent Rendering 确保高频数据更新下的界面响应。
- **Tailwind CSS**：极致的响应式与现代化 UI 视觉体验。
- **Lucide Icons**：简洁一致的图标指引。
- **TronGrid API**：标准波场区块链底层数据服务。

## 本地部署

### 1. 环境准备

确保您的系统中已安装 [Node.js](https://nodejs.org/) (建议 v18.0.0 或更高版本)。

### 2. 安装依赖

在项目根目录下打开终端，执行以下命令安装必要的开发依赖：

```bash
npm install
```

### 3. 启动开发服务器

执行以下命令启动实时预览服务：

```bash
npm run dev
```

启动后，默认可通过浏览器访问 `http://localhost:5173` (具体端口见终端输出)。

### 4. 生产环境构建

如果您需要构建部署到生产环境，请执行：

```bash
npm run build
```

## 配置说明

1. **API Key 配置**：
   - 启动应用后，点击右上角的 **“设置”** 图标。
   - 输入您的 **TronGrid API Key**（可在 [Trongrid.io](https://www.trongrid.io/) 免费申请）。
   - 点击保存，系统将自动开始同步主网数据。

2. **采样规则自定义**：
   - 应用内置了常用采样间隔（1/20/60/100）。
   - 您可以点击 **“+ 规则”** 新增自定义规则，单独设置该规则下的 **走势行数** 与 **珠盘行数**。

## 分析逻辑

- **单双 (Parity)**：
  - **单 (ODD)**：哈希末位数字为 1, 3, 5, 7, 9。
  - **双 (EVEN)**：哈希末位数字为 0, 2, 4, 6, 8。
- **大小 (Size)**：
  - **大 (BIG)**：哈希末位数字 5 - 9。
  - **小 (SMALL)**：哈希末位数字 0 - 4。
