
# 波场 (TRON) 哈希走势实时分析大盘 (高级版)

这是一个专为波场区块链设计的实时哈希数据可视化工具。它通过连接 TronGrid API，实时捕获区块哈希，并自动分析哈希末尾数字的“单双”与“大小”走势。

## 核心功能

- **全中文本地化**：深度汉化所有 UI 元素与提示。
- **3秒实时同步**：毫秒级监听波场主网出块。
- **高级自定义采样**：
  - **自定义名称**：为不同的分析策略命名（如“快速趋势”、“长线波动”）。
  - **自定义间隔**：自由设定 1, 10, 50, 200 等任意出块采样频率。
  - **自定义起始区块**：支持锁定起始高度，方便精确对齐特定时间点或合约事件。
- **瞬时切换技术**：内存级主数据集缓存，不同采样规则间来回切换数据 0 丢失，无感补齐。
- **高性能架构**：React 19 + Tailwind CSS，支持长时间在线运行，内存占用低。

## 运行配置

### 1. 环境要求
- 现代浏览器 (Chrome, Safari, Edge)。
- TronGrid API Key (必填)。

### 2. 获取 API Key
- 前往 [TronGrid.io](https://www.trongrid.io/) 免费注册并创建 API KEY。

## 本地部署

1. **克隆/下载**代码至本地目录。
2. **启动环境**：
   ```bash
   # Python 环境下
   python -m http.server 8000
   ```
3. **访问地址**：浏览器打开 `localhost:8000`。
4. **接入配置**：点击右上角设置图标，粘贴 API Key。

## 分析逻辑
- **单双**：哈希末位数字为 1,3,5,7,9 为单，0,2,4,6,8 为双。
- **大小**：哈希末位数字 0-4 为小，5-9 为大。
